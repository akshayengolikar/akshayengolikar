### 📰 BBC News Classification with CNN-LSTM Fusion
> Deep learning model for multi-category news article classification — *University of Dayton*
- 🧬 Designed a **hybrid CNN-LSTM architecture** achieving **98.89% accuracy**, outperforming standalone models
- 📝 Preprocessed raw text using tokenization, lemmatization, and noise removal
- 🔤 Implemented **Word2Vec embeddings** to capture semantic relationships in text
- ⚙️ Optimized hyperparameters (batch size, learning rate, embedding dimensions) for peak performance
- 📊 Evaluated against standalone CNN and LSTM baselines, demonstrating significant improvements

`Python` `TensorFlow` `Scikit-learn` `NLP` `Deep Learning`
